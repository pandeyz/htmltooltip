All About HTML ToolTip

What is tooltip?

The tooltip or infotip or a hint is a common graphical user interface element. It is used in conjunction with a cursor, usually a pointer. The user hovers the pointer over an item, without clicking it, and a tooltip may appearâ€”a small "hover box" with information about the item being hovered over.

What's new about this?

There are so many plug-ins available to show tooltip on your webpage in which one can show text of any length on tooltip. But as the name suggest you can show text as well as html like image and anchors in tooltip.

How this is possible?

This is possible by just including a light weight fully configured jquery library with your code and just call its functionality and its done!

Include the following file in script tag: htmltooltip.js

< script type = "text/javascript" src = "htmltooltip.js" > script >

How to configure?

You can configure all of its attributes by just passing its value at the time of initialization.

Initialization: 
                    

                        $("#p1").htmltooltip();
Initialization with attributes: 
                    

                        $("#p1").htmltooltip({
                            autoClose: false,
                            backgroundColor: "#F2F2F4",
                            borderRadius: "5px",
                            bottom: "-20px",
                            color: "black",
                            fontSize: "12px",
                            left: "0px",
                            padding: "5px",
                            right: "0px",
                            top: "0px"
                        });
  • autoClose: The tooltip is automatically hide when mouse out is triggered, its default value is true. If set to false
  • backgroundColor: Background color of the tooltip. You can pass color using Color Name like RED or HEX Value like #F0F8FF
  • borderRadius: The border radius of the tooltip
  • bottom: To set the tooltip bottom property
  • color: Color of the text in tooltip. You can pass color using Color Name like RED or HEX Value like #F0F8FF
  • fontSize: Font size of the text in tooltip. You can pass any value like 12px, 14px etc
  • left: To set the tooltip left property
  • padding: Padding of the tooltip. You can any value like 5px, 8px etc
  • right: To set the tooltip right property
  • top: To set the tooltip top property
